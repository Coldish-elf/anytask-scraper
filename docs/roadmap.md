# Roadmap

## 0.9.х

- Добавить команду `serve` в `docs/CLI.md` как отдельную секцию.
- Документировать вкладку DB в `docs/TUI.md`.
- Скринкасты для README и документации.
- Экспорт `__version__` из `__init__.py` (динамически из `importlib.metadata`).

## 0.10.0

- Тесты без фикстур
- Установка оценки через issue-страницу.
- Смена статуса задания (зачтено/на доработку/на проверке).
- Добавление комментария к submission.
- CLI-команды: `anytask-scraper push grade`, `push status`, `push comment`.
- API-эндпоинты: `POST /submissions/{issue_id}/grade`, `/status`, `/comment`.
- Интеграция в TUI: кнопки действий в деталях submission.
- Diff между снимками очереди: показывать что изменилось с последнего sync.
- Статистика: сколько новых/проверенных/на доработке за период.
- Планировщик: периодический `db sync` по расписанию (cron-like).
- CLI-команда `watch`: мониторинг очереди в реальном времени с уведомлениями в терминале.

## 1.0.0

- Публикация на PyPI.
- GitHub Actions CI/CD.
- CONTRIBUTING.md.
- Разбить `main.py` на отдельные модули.
- Поддержка мультиаккаунтов.
- Аутентификация API-запросов.
- Rate limiting.
- README на английском языке.
- Документация на двух языках.
- Прямая выгрузка ведомости в Google Sheets через API.
- Docker-образ: готовый контейнер с API-сервером для деплоя.
